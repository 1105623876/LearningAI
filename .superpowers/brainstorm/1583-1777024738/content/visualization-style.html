你希望用什么样的方式来可视化这些概念？

选择最能帮助你理解的可视化风格

A

交互式动画演示

像 3Blue1Brown 那样的动画，可以拖动参数、逐步播放，看到注意力权重如何计算、梯度如何流动

技术栈：React + D3.js / Three.js
B

静态图解 + 代码对照

精美的静态图示配合代码高亮，左边是图解右边是代码，点击图中的部分高亮对应代码行

技术栈：React + SVG + Monaco Editor
C

Jupyter Notebook 风格

可执行的代码单元格 + 可视化输出，修改代码立即看到效果，像 Distill.pub 那样的交互式文章

技术栈：JupyterLab / Observable / React + Pyodide
D

混合模式

根据知识点特性选择：简单概念用静态图解，复杂机制用交互动画，算法用可执行代码

灵活但开发量大
